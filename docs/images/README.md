# Screenshots for the main README

The repo ships **placeholder** PNGs so the readme always shows two panels. **Replace them** with real captures when you can.

Target files in this folder (overwrite the placeholders):

| File | What to show |
|------|----------------|
| `windows-125-without.png` | Same app, **without** `AppScaler` — typical layout/overflow/portal issues at **125%** (or 150%) **Windows display zoom**. |
| `windows-125-with.png` | **Same** app and **same** OS zoom, **with** `<AppScaler>` — coherent full layout. |

**Tips**

- Same browser width, same page, same theme — only the scaler on/off should change.
- Capture the **viewport** (not the whole monitor) so the comparison is readable in the readme.
- Optional: add a thin caption in the image (e.g. “125% display zoom”) for clarity.

After exporting, name the files exactly as above and commit them to the repo.
